### 开发环境和技术栈
 - 操作系统：window 10 64位
 - 开发工具：sublime
 - 前端：vue + vue-router + axios

### 运行项目
1. 下载项目
2. 安装依赖 `npm install`
3. 启动项目 `npm run dev`
4. 访问`http://localhost:8082`
